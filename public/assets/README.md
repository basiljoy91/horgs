# Herbs Atelier Asset Kit

This folder contains the first curated visual asset pack for the luxury herbal
site direction.

Included:

- `photos/`
  - botanical still-life style hero art
- `objects/`
  - ceramic vessels and tabletop compositions
- `overlays/`
  - transparent leaf overlays for layered sections
- `textures/`
  - grain, paper, and shadow textures
- `motion/`
  - subtle animated SVG loops for ambient depth

Notes:

- These assets are handcrafted SVG-based placeholders designed to match the
  final art direction immediately inside the product.
- They are intended as premium stand-ins until we replace or supplement them
  with real photographic assets or AI-generated imagery.
- If you want true photoreal botanical photography next, ask explicitly for the
  image generation fallback workflow and I can guide that path separately.
